# Documentation

> This documentation collects the practical guides for installing, using, extending, and troubleshooting the profile.

## Start Here

- **Install or restore the profile:** follow the [Installation guide](installation.md).
- **Find an available command:** browse the [Function catalog](functions.md).
- **Fix profile loading problems:** work through [Troubleshooting](troubleshooting.md).

## Usage Notes

- **Load functions in remote sessions:** see [Remote sessions](remote-sessions.md).
- **Keep machine-local helpers private:** use [Local functions](../Functions/Local/README.md).
- **Contribute tests or new functions:** start with [Tests](../Tests/README.md) and [Contributing](../README.md#contributing).
